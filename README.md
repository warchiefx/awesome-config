# WCX's AwesomeWM setup

Taken from several places around the net.

Required: AwesomeWM 4.2+

## Setup

- Replace your .xinitrc with start-awesome.sh
- Point .xsession to your .xinitrc

- volumeicon-alsa
- lxappearence
- slack
- telegram-desktop
- chromimu/firefox
- autorandr
